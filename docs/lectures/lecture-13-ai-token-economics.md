# Lecture 13 — AI Topic II: Token Economics & Project ROI

!!! info "Session Info"
    **Date:** Monday, 2026-06-15 &nbsp;·&nbsp; **Time:** 16:20–18:10 &nbsp;·&nbsp; **Room:** YF108
    **Instructor:** Dr. Zhijiang Chen

## Learning Objectives

1. Decompose the **cost structure of an LLM-powered system**: input / output / cached tokens, infrastructure, observability.
2. Quantify the **multiplier effect of agents** (3–10× the calls of a basic chatbot).
3. Apply **prompt caching, semantic caching, and KV caching** to lower production cost by 40–90%.
4. Build a **payback-period analysis** for an AI initiative.
5. Synthesize the course: when to apply which methods.

## Topics

- 2026 LLM API price landscape (Claude, GPT, Gemini, Chinese-market models).
- Agent call patterns: planning, tool calls, verification, retries.
- Caching strategies and their measured economic impact.
- Empirical payback periods: customer service ≈ 4 months; engineering ≈ 9 months.
- **Final-exam review** — key formulas, common mistakes, sample problems.

## In-Class Exercise

For a hypothetical "AI legal assistant" SaaS with 8,000 monthly users, compute monthly token spend with and without prompt caching, and find the break-even subscription price.
