# Lecture 8 — Cost Estimation I: Size Metrics & Function Points

!!! info "Session Info"
    **Date:** Wednesday, 2026-06-10 &nbsp;·&nbsp; **Time:** 16:20–18:10 &nbsp;·&nbsp; **Room:** YF302
    **Instructor:** Dr. Zhijiang Chen

## Learning Objectives

1. Compare **size-based metrics**: LOC, Function Points (IFPUG), Use-Case Points.
2. Compute **Unadjusted Function Points (UFP)** from a small specification.
3. Apply **Value Adjustment Factor (VAF)** to obtain **Adjusted Function Points**.
4. Recognize common estimation biases and how to counter them.

## Topics

- Why size estimation is hard but unavoidable.
- LOC: when it helps, when it misleads.
- Function Points: external inputs / outputs / inquiries, internal logical files, external interface files.
- VAF and the 14 General System Characteristics.
- Use-Case Points and other modern variants.
- Estimation biases: anchoring, optimism, planning fallacy.
