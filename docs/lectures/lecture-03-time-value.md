## Learning Objectives

1. Explain *why* money has time value.
2. Compute **Present Value (PV)** and **Future Value (FV)** for single cash flows.
3. Distinguish **simple**, **compound**, **nominal**, **effective**, and **continuous** interest rates.
4. Convert between rate conventions across compounding periods.

## Key Formulas

- **Future value of a single sum:** $FV = PV \cdot (1 + i)^n$
- **Present value of a single sum:** $PV = \dfrac{FV}{(1 + i)^n}$
- **Effective annual rate** from nominal rate $r$ compounded $m$ times per year: $i_{\text{eff}} = (1 + r/m)^m - 1$
- **Continuous compounding:** $FV = PV \cdot e^{rn}$
