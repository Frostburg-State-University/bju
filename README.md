# Software Engineering Economics

Course site for **Software Engineering Economics** taught by **Dr. Zhijiang Chen** at **Frostburg State University**, Summer 2026.

- Live site: <https://frostburg-state-university.github.io/bju/>
- Course schedule, lecture notes, exercises, and assessment artifacts are all here.

## Local Development

```bash
pip install -r requirements.txt
mkdocs serve
```

Then open <http://127.0.0.1:8000>.

## Deployment

The site is built and published to GitHub Pages via the workflow in `.github/workflows/deploy.yml` on every push to `main`.
