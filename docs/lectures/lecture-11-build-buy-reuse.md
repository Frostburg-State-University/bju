# Lecture 11 — Build / Buy / Reuse & Value-Based Software Engineering

!!! info "Session Info"
    **Date:** Saturday, 2026-06-13 &nbsp;·&nbsp; **Time:** 16:20–18:10 &nbsp;·&nbsp; **Room:** SY109
    **Instructor:** Dr. Zhijiang Chen

## Learning Objectives

1. Frame **Build vs. Buy vs. Reuse vs. Open Source vs. SaaS** as an economic decision.
2. Apply a **decision matrix** with weighted criteria.
3. Introduce **Value-Based Software Engineering (VBSE)** and the stakeholder value proposition.
4. Map technical alternatives onto a **value vs. cost** plane.

## Topics

- Reuse economics: COCOMO II reuse model, NIH ("not invented here") bias.
- Decision matrix construction and weighting.
- VBSE: stakeholders, value propositions, win conditions.
- "Earned Value" vs. classical earned-value management.

## In-Class Exercise

For a search/authentication/payment component, build a decision matrix comparing Build, Buy (SaaS), and Open Source. Identify the most decision-relevant criterion.

## Homework

- Apply VBSE to your project: list stakeholders, value propositions, and points of conflict. Provide a recommended resolution.
