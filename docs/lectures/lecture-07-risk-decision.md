# Lecture 7 — Risk & Decision under Uncertainty

!!! info "Session Info"
    **Date:** Tuesday, 2026-06-09 &nbsp;·&nbsp; **Time:** 16:20–18:10 &nbsp;·&nbsp; **Room:** YF108
    **Instructor:** Dr. Zhijiang Chen

## Learning Objectives

1. Distinguish **risk** (known probabilities) from **uncertainty** (unknown probabilities).
2. Build and evaluate a **decision tree** with expected-value rollback.
3. Apply a **utility function** to capture risk attitude.
4. Set up a basic **Monte Carlo simulation** for an NPV model.

## Topics

- Probability primer for engineers.
- Decision trees: nodes, branches, rollback.
- Expected Monetary Value (EMV) and Expected Utility.
- Monte Carlo simulation in a spreadsheet or Python.
- Risk in software-economics decisions: schedule, scope, vendor lock-in, model drift.

## In-Class Exercise

Decide whether to launch an AI feature with three uncertain outcomes (high adoption / medium / low). Build a decision tree and recommend the action under both risk-neutral and risk-averse utility.

## Homework

- Code a small Monte Carlo simulation for one of your project's NPV models. Plot the NPV distribution; report P10/P50/P90.
